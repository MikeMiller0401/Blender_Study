# Blender_Study
blender的学习笔记

`：视角切换pie目录

x: 删除物体

shift + A: 新建物体

G: 移动；G + X/Y/Z: 限定X/Y/Z移动；按住crtl可以以方格为单位进行移动

S：缩放；S + X/Y/Z: 限定X/Y/Z缩放

R：旋转；R + X/Y/Z: 限定X/Y/Z旋转

Alt + G/S/R：取消移动/缩放/旋转

H：隐藏选中的物体

Shift + H：隐藏选中以外的物体

Shift + D：移动并复制

工作区边缘出现+号之后朝外面拖动可以复制该工作区，朝内可以删除工作区

Shift + S：游标pie目录

Shift + C：游标恢复到世界中心

shift + 右键： 移动游标

table:切换工作模式

## 编辑模式下：
1：选择点

2：选择线

3：选择面

### 十大建模操作：

E：挤出

I：向内挤出

Ctrl+B:倒角

Ctrl+R:循环切割

M：合并

V：断开

F：填充；Ctrl+F栅格填充

K：切刀

Ctrl+E：桥接；Ctrl+J：合并物体

P：分离
